# KoThumb2 - 画像一括リサイズ・変換ツール

KoThumb2は、大量の画像をドラッグ＆ドロップで素早くリサイズ・変換するためのデスクトップアプリです。
SNSへの投稿用や、ブログ・資料作成などで画像サイズをサッと揃えたい時に最適です。

## ✨ 特徴

- **かんたん操作**: フォルダを窓に放り込む（ドラッグ＆ドロップ）だけで準備完了。
- **とにかく高速**: マルチコアCPUをフル活用した並列処理により、数百枚の画像もあっという間に処理します。
- **きれいに縮小**: 高品質なリサイズエンジン（ImageSharp）により、縮小しても画像がボケにくいです。
- **かしこいリネーム**: 「元の名前の頭に _resized_ を付ける」「連番（001, 002...）にする」など、自由なルールを設定できます。
- **WebP対応**: 次世代形式の WebP への変換もバッチリ対応しています。

## 📸 主な機能

- **リサイズ**: 幅指定、高さ指定、長辺基準（縦横関係なく長い方を合わせる）の3パターン。
- **形式変換**: JPG, PNG, WebP への一括変換。
- **サブフォルダ対応**: フォルダの中にフォルダがあっても、そのままの構造で出力できます。
- **ピンポイントプレビュー**: 処理ログのファイル名をクリックすると、その画像の「変換後イメージ」をすぐに確認できます。
- **安全設計**: 入力フォルダと出力フォルダに親子関係がある場合は、無限ループ防止のため実行をブロックします。
- **最前面表示**: チェック一つで、他のアプリを操作中も常に手前に表示させておくことが可能です。

## 🚀 使い方

1. **フォルダを入れる**: 処理したい画像が入ったフォルダを、アプリのウィンドウにドラッグ＆ドロップします。
2. **設定を選ぶ**: 
   - 「リサイズ設定」でサイズを決めます。
   - 「出力フォーマット」で保存したい形式（JPGなど）を選びます。
3. **名前を決める**: 「ファイル名設定」で保存する名前のルールを確認します。画面下のプレビューで結果を予測できます。
4. **開始！**: 右下の「実行開始」ボタンを押すと処理が始まります。
5. **確認**: ログから気になるファイルをクリックして仕上がりを確認したり、終了後に「出力先を開く」で結果をチェック。

## 💻 動作環境

- **OS**: Windows 10 / 11

### 配布形式について

KoThumb2は2つの配布形式を提供しています：

#### 📦 .NET同梱版（推奨）
- **ファイル名**: `KoThumb2-win-x64-standalone.exe`（または類似の名前）
- **特徴**: .NETランタイムが実行ファイルに含まれているため、追加のインストールは不要です
- **サイズ**: 大きめ（約100MB以上）
- **メリット**: ダウンロードしてすぐに使える、.NETのバージョン競合の心配がない
- **推奨対象**: 初めて使う方、.NETをインストールしたくない方

#### 🔧 .NET非同梱版（軽量版）
- **ファイル名**: `KoThumb2.exe`
- **特徴**: 実行に.NETランタイムが別途必要です
- **サイズ**: 小さめ（数MB程度）
- **必須コンポーネント**: [.NET 9.0 デスクトップ ランタイム](https://dotnet.microsoft.com/download/dotnet/9.0)
  - ※起動時にエラーが出る場合は、上記リンクより「Run desktop apps」の「x64」版をインストールしてください
- **メリット**: ファイルサイズが小さい、複数の.NETアプリで共通のランタイムを使用
- **推奨対象**: すでに.NET 9.0がインストールされている方、ファイルサイズを抑えたい方
